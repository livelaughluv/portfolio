>GitHub Notes

This year, we took notes using GitHub. Each student created programs (I used Python on Replit) that allowed users to choose which notes they would like to read. Once they typed in their option, the users could see the notes for that section. Some sections had sub-sections that required users to choose another option. Click this button to access the GitHub repository where I stored my notes.

Top Notes

While I took notes on all of the lessons we had this quarter, I'll summarize my top three lessons here.

Starting Systems (8/30/22)

This lesson was about systems and system changeovers. It was one of my favorite lessons I took notes on because we had already started learning about the subject last year (IB Comp Sci Year 1), but we began to go more in depth this year. I enjoyed reviewing what I already knew and learning new things at the same time.

System Design (9/6/22)

This lesson was about, as you probably guessed by the title, types of system designs. The content was not what I truly enjoyed about this lesson (not that it wasn't enjoyable, just that I chose this as one of my top three lessons for another reason), it was that while coding the notes for this lesson I first extended my code to include extra sub-sections and choices for the user to learn about. It was a bit difficult and broke my brain at first, but I was very happy once I got it to work.

Ethics and Impacts of IT (9/26/22)

This lesson was about the ethics and impacts of IT (no duh). This lesson is one of my top three lessons because I was able to connect Computer Science to another IB course, Theory of Knowledge. TOK is very interesting for me, and I enjoyed exploring the TOK side of Computer Science.
